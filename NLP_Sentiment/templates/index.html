Project Overview

This project analyzes mobile device reviews using a hybrid approach, combining Supervised NLP for sentiment prediction and Unsupervised Learning for topic discovery.

🧹

Data Preprocessing

Raw text is cleaned using NLTK (stop-word removal & regex formatting).

🔢

TF-IDF Vectorization

Text is converted to numerical matrices representing word importance.

🤖

Classification Models

Random Forest & Logistic Regression models are trained to predict Positive/Negative sentiment.

📊

Latent Dirichlet Allocation (LDA)

Used to perform Topic Modeling, extracting underlying themes and keywords across all reviews.

🎯

K-Means Clustering

Groups similar mobile reviews into distinct clusters to automatically discover user feedback patterns.
